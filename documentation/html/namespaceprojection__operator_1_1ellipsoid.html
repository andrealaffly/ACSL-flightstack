Convex function to compute h and dh_dx. The outer convex set is an ellipsoid centered in x_e with semi-axis length described by the diagonal elements that populate the S matrix. If S = diag(1/a^2, 1/b^2, 1/c^2) then the semi-axis lengths are [a, b, c]. The inner convex set is the outer convex set ellipsoid scaled by the coefficient alpha = 1/sqrt(1 + epsilon).
